關於 VCC Clipper

VCC Clipper 是一套內網跨子網 P2P 協作工具，透過 WebRTC 建立點對點連線，

版本v1.0.0
作者RTHKVCC Team
技術WebRTC · WebSocket · Python
授權MIT

功能

  • 💬 聊天室 — 即時訊息廣播
  • 📁 檔案傳輸 — 拖放上傳，支援大檔案

已知限制

  • 需要 Python 3.8+ 執行信令伺服器
  • 跨子網需要網路設備支援 UDP 路由
  • 聊天室訊息傳送需所有用戶保持連線
